Welcome to SDUThesis!

SDUThesis 是山东大学学位论文 LaTeX 模板。你还可以访问：

  • 开发主页；
  • CTAN 主页。

在使用中有任何问题、意见或建议，你都可以在项目主页上提交 Issue。

How to use?

安装

如果你安装了完整的 TeX Live 或者完整的 MiKTeX（包括 ProTeXt），那么你已经安装好了 SDUThesis，可以直接使用。

否则，你可以：

  • TeX Live 用户执行命令 tlmgr install sduthesis；
  • MiKTeX 用户在 MiKTeX Package Manager 中检索 sduthesis 并安装。

如果希望「开箱即用」，你也可以遵照如下步骤：

  • 安装好一个 TeX 发行版（通常是 TeX Live、MiKTeX 或者 MacTeX）；
  • 在本页顶部下载最新的安装包，并解压；
  • 执行其中的 build 脚本（Windows 用户可以直接双击 build.bat）。

说明手册

模板有一个说明手册（sduthesis.pdf）以及一个用模板排版的示例文档（sduthesis-demo.pdf）。对于熟悉 LaTeX 的用户，阅读这两个文档应当能够排版出相应的论文。

如果你对 LaTeX 不熟悉，那么建议你阅读这篇，以及下载这个压缩包。若有其他疑问，请先阅读这篇，然后到 CTeX 论坛提问。

You are WANTED!

目前，该模板已上传至 CTAN，并被 TeX Live 和 MiKTeX 收录。这意味着，这两个 TeX 系统的用户，可以很容易地安装、更新和使用这份模板。

由于我已经毕业并参加工作多年，已无有精力维护模板。故此，我希望能有「新鲜血液」参与到模板维护中来。基本要求只有两条：

  • 您是山东大学在读学生或在职教师；
  • 您了解并使用过 LaTeX。

具体的参与方式有三种：

  • 【初级】试用模板，在 GitHub 上提交 issue，帮助项目改进。
  • 【中级】Fork 项目，添加自己的修改，而后以 Pull Request 的方式将修改合并至主项目。
  • 【高级】以 collaborator 的身份加入项目，参与代码的编写、主页的维护、Pull Request 及 Issue 的管理。熟悉 LaTeX 及其宏编程的用户，欢迎以这种方式参与并传承下去。

若您有意参与维护，请毫不犹豫地联系我。: )
